# W3Admin - Free W3.CSS admin template.

W3Admin is a fully responsive W3.CSS admin dashboard template. It is built with Vue 2.6 framework, HTML5, and CSS. It works on all major web browsers, desktop, and all smartphone devices. It is very easy to customize and developer friendly template. It can be used for any type of web applications eCommerce dashboard, custom admin panel, project management admin, CRM, CMS, ERP, etc.
